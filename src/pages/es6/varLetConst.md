# var let const

## 区别

|                                                                   | var                  | let | const |
| ----------------------------------------------------------------- | -------------------- | --- | ----- |
| 变量提升(变量可以在声明之前调用)                                  | 存在,值为`undefined` | F   | F     |
| 暂时性死区(只有等声明变量的那一行代码出现,才可以获取和使用该变量) | F                    | T   | T     |
| 块级作用域                                                        | F                    | T   | T     |
| 重复声明                                                          | T                    | F   | F     |
| 修改声明的变量                                                    | T                    | T   | F     |
